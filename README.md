# Multi-Tenant Web-Based Accounting Platform

This project is designed for agentic AI development. The project structure and documentation are optimized for AI agents to build and maintain the platform.

## Project Structure

*   `backend/`: Source code for backend microservices.
*   `frontend/`: Source code for the web-based user interface.
*   `docs/`: Detailed specifications and documentation.
*   `CLAUDE.md`: The constitution for AI agents.
*   `STATUS.md`: Project change log.
*   `TASKS.md`: Development roadmap and task tracking.
*   `.claude/hooks/`: Automation scripts for agentic development.

## Getting Started

To begin development, provide an AI agent with the context of this repository. The agent will automatically read `CLAUDE.md`, `STATUS.md`, and `TASKS.md` to understand the project's current state and goals.
